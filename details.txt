Dataset Link: https://www.kaggle.com/alxmamaev/flowers-recognition
Techniques used: 
- Transfer learning with pretrained ResNet50 model
- Feature Selection using Evolutionary Optimization Algorithms - Genetic Algoritm (GA)
- Model was trained on Google Colab.
Accuracy:
- 94.34% using Transfer Learning
- 95.26% after Feature Selection using Optimization GA.
